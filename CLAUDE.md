## Style Constraints (The Constitution)

- **Colors (Intellectual Glamour)**: 
  - Muted Black (#111111) — The Ground / Void.
  - Seashell (#EAE4DA) — Primary Text / Canvas.
  - Mustard (#E3B044) — Insight / Accent 01.
  - Lavender (#D9D2E9) — Context / Accent 02.
  - Sky (#3A9EA4) — Structure / Accent 03.
  - Tea (#4F6D7A) — Foundation / Accent 04.
  - Tangerine (#FA7714) — Transformation / Accent 05.
  - Red Passion (#F72658) — Insight (Peak) / Accent 06.
- **Lines**: 0.5px–1.5px only (2.0px for Red Passion focal only).
- **Motion**: Everything draws itself. No typewriter text. Stagger everything.
- **Font**: Instrument Serif (Display Italic), DM Sans (Body), JetBrains Mono (Metadata).
